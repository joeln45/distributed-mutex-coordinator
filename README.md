# Distributed Mutex Coordinator

A socket-based **centralised Distributed Mutual Exclusion (DME)** system in Java, implementing token-passing coordination between a central `Coordinator` and multiple `Node` processes.

> 🚧 This README is a placeholder. A full version with architecture diagrams, sample logs, and run instructions will be added once the project is fully modernised.

## Origin

Originally built as the **CSCU9V6 Distributed Systems** assignment at the University of Stirling. Now being upgraded into a portfolio-grade project with Maven, JUnit 5, SLF4J/Logback, and GitHub Actions CI.
